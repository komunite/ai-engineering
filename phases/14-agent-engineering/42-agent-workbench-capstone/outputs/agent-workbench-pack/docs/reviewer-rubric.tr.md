# Reviewer Rubric

Beş boyut, 0'dan 2'ye skorlanır.

1. Problem uyumu — değişiklik görevi belirtildiği gibi çözdü mü?
2. Scope disiplini — düzenlemeler kontrata sınırlı mıydı?
3. Varsayımlar — gizli varsayımlar yazıldı mı?
4. Verification kalitesi — kabul gerçekten hedefi kanıtlıyor mu?
5. Handoff hazırlığı — sonraki session temizce devralabilir mi?

Toplam >= 7 sıfır olmadan: pass. Toplam 5-6: soft fail. 5'in altı veya herhangi bir sıfır: hard fail.
